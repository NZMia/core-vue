Vue project for CORE
------
This is for Vue project . 
Local port; 9091

#### Quick start
--------
##### Clone
```
  git@github.com:NZMia/core-vue.git
```
##### Install
```
  npm install
```
##### Dll
_The vendor needs to be packaged once when the first time running. Repackage it if dependencies change._
```
  npm run dll
```
##### Build
```
  npm run build
```
##### Start
```
  npm run Start
```
